This is a simple script that will cycle through RGB spectrum on all phillips hue lights and corsair components compatible with ICUE.
This should allow for a big RGB setup in a room where lights and corsair components are present

Uses phue (link: https://github.com/studioimaginaire/phue)
Uses ICUE SDK (link: https://github.com/CorsairOfficial/cue-sdk-python)

If first time running, add "b.connect()" to script. Then press the hue bridge button and within 30 seconds, run the script
